# Типы данных

Перечисление `TDocVariantKind` (`dvUndefined`, `dvObject`, `dvArray`) — способ в скрипте отличить JSON-объект от JSON-массива и от неинициализированного значения. Платформа активно использует JSON как носитель структур: аргументы журналов, полезная нагрузка сообщений, параметры шаблонов страниц. Перед обходом элементов через `_Count` / `Name` / `Value` принято читать `_Kind` и ветвить логику: иначе объект и массив обрабатываются одинаково ошибочно. Этот тип — ключ к разделу работы с JSON-объектами.
